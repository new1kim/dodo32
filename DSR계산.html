💡 DSR 계산기 이용 안내
⚠️ 필독 지침
항상 첫 번째 입력칸이 상담받으실 '본건 대출금액'이어야 합니다. 그래야 월별 부담금액 및 최대한도, 신DTI(180개월) 계산이 정상적으로 연산되어 표시됩니다.
✨ DSR 40%, 50% 최대한도 금액을 터치하시면 해당 한도 금액이 본건 대출원금 칸에 자동으로 입력됩니다.
📊 대출현황 입력 시 참고사항
  • 신용대출: 5년 (60개월) 입력
  • 비주택담보대출: 8년 (96개월) 입력
  • 전세대출 / 예적금담보대출: 만기일시 형태이므로 [이자만] 체크
  • 유가증권담보: 8년 (96개월) 입력
  • 캐피탈 / 카드대출 / 학자금 등: 12개월 기준으로 1년간 실제 총 납입금액을 계산하여 입력
⚙️ 숨겨진 편의 기능 (롱프레스)
  • 장래예상소득 체크박스를 3초간 누르면, 은행별 차이나 새로운 공지 요율에 맞춰 직접 요율 수정이 가능합니다.
  • [계산제외] 체크박스를 2초간 누르면 해당 대출 라인이 즉시 삭제됩니다.
  • 본건 대출행의 금리, ST금리, 개월수 칸을 3초간 누르면 기본 시작 세팅값을 저장해둘 수 있습니다.
ℹ️ 기타 안내
  • ST금리 구분 이유: 스트레스 금리는 DSR 계산식에만 포함하고, 실제 월 부담액(이자) 계산에서는 제외하기 위함입니다.
  • 계산제외 버튼: DSR 연산에서 아예 제외하거나 상환 조건부 상담 시 체크하는 용도입니다.
  • 신DTI 계산값: 신DTI 계산값은 항상 180개월로 계산되어있음, 그 외는 입력값에 따라 다름. 거치 적용시 180개월 내에서도 해당 기간만큼 거치기간이 반영됩니다.
